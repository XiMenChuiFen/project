# vue-word

* 内部使用的单词管理程序

#### 运行

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

#### 技术点

* 翻译使用有道翻译
* 发音使用百度发音
* UI使用ElementUi
